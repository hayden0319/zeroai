## 核心竞品全景分析
### 竞品分类与核心格局
📌 **核心结论**：AI 聚合平台分为**开源自建类、企业级 API 聚合类、多模态 AIGC 聚合类、智能体 + 聚合类**四大赛道。竞争焦点在**部署便捷性、模型覆盖、运营工具、合规性、多模态能力、API 灵活性**。初创机会：聚焦「官方/中转 API 双适配 + 快速私有化 + 智能体 RAG + 多模态模板 + 基础合规 + 成本可视化」的MVP解决方案，避开大厂资源竞争，快速验证市场空白（单一能力短板）。

### 痛点与核心机会
#### 核心痛点（初创视角简化）
1. **部署**：开源门槛高，SaaS 无私有化，数据风险。
2. **功能**：智能体与多模态割裂，模板/合规弱。
3. **运营**：技术平台无商业闭环，ToC 无企业工具。
4. **成本**：不透明，无管控，多切换重复付费。
5. **API**：单一模式，配置繁琐，限流无备选。

#### 核心机会（增强初创策略）
整合GoAmzAI部署 + BuildingAI智能体 + SparkAI多模态 + OneAPI管理优势，打造双API兼容MVP。初创差异化：低成本开源起步（Apache 2.0），快速MVP验证（先ToC，后企业），用户反馈迭代；融资卖点：市场空白填补，全栈但精益（优先P0），预计种子轮估值基于用户增长。

## 产品需求文档（PRD）
### 产品概述
#### 核心定位
面向个人/中小团队（初创MVP焦点），渐进企业，提供「官方 API Key + 中转 API 双对接、多模型聚合、多模态创作、智能体编排、私有化部署、基础运营、合规」的服务。解决切换繁琐、配置复杂、安全顾虑、变现难、成本控的核心痛点。初创优化：MVP聚焦SaaS版，私有化作为P1选项，强调快速上线验证市场。
#### 核心目标（SMART + 初创调整）
* 🎯 **业务**（上线3个月）：累计用户1K+，付费转化15%+，核心场景留存40%+；种子融资目标：基于MVP数据，月营收目标5K+。
* ⚙️ **技术**：并发QPS≥1000（初创规模），响应≤500ms，可用性≥99%；双API切换100%，延迟误差≤150ms。
* 🚧 **边界**：不自研模型，聚焦聚合+场景+运营+合规+灵活API；SaaS先上，私有化后跟；数据本地化，合规优先。
#### 功能优先级界定（精益化）
| 优先级 | 说明 | 核心功能清单 |
| ---------- | ------------- | ------------------------------------------------------------------ |
| P0（MVP必做） | 基础可用，快速验证 | 多模型接入/切换、双API对接（简化配置）、多模态入口、基础计费、安全、SaaS用户体系 |
| P1（1-3个月） | 增强体验，商业闭环 | 智能体/RAG、模板中心、成本管控、基础运营（会员/支付）、团队协作、负载均衡 |
| P2（3-6个月） | 拓展，视融资 | 自定义接入、API开放、行业方案、迁移、高级审计 |

### 用户画像与核心场景
#### 核心用户画像（聚焦初创目标群）
| 用户类型 | 核心特征 | 核心需求 | 核心痛点 |
| --------------------- | ----------------------- | ----------------------------------- | ------------------------ |
| 个人创作者 | 预算低、操作简单、多模型对比 | 多模态创作、低成本、易上手 | 切换繁琐、付费不透明 |
| 中小团队 | 协作、成本控、统一入口 | 额度分配、共享模板、统计 | 无协作、成本难控 |
| 企业用户（P1后） | 安全、合规、私有化 | 脱敏、本地部署、审计 | 泄露风险、无私有化 |

#### 核心使用场景（简化至MVP关键）
1. **个人创作**：选择模板→填充→一键生成（GPT+Midjourney）→编辑导出。
2. **团队协作**：创建团队→分配额度→共享记录→查看报表。
3. **企业合规**（P1）: 私有化部署→脱敏调用→审计导出。
4. **智能体**（P1）：拖拽编排→关联知识库→分发微信。
5. **官方API**：配置多Key→负载均衡→低成本调用。
6. **中转API**：一键启用→查看明细→付费。

### 核心功能性需求
#### 多模型接入与切换
覆盖8+主流（GPT/Claude/豆包等）；双模式支持；一键切换+对比；收藏默认；实时监控+自动切换。
#### 统一多模态交互中心
| 类型 | 功能 | 指标 |
| ---- | ----------------------------------------- | ------------------------------ |
| 文字 | 长输入、记忆、导出 | ≤500ms，PDF/TXT |
| 绘画 | 文/图生图、模板 | 1080P≤45s |
| 视频 | 文/图生视频 | 720P≤5min |
| 语音 | 转文字/语音 | 准确≥95% |
| 文档 | 上传/问答 | ≤50MB，≤15s |

#### 场景化模板中心
30+模板（创作/办公/开发）；搜索/自定义/上传（审核+分成10%）。
#### 智能体编排与 RAG（P1）
可视化拖拽；知识库导入/检索≥90%；分发微信/H5+统计。
#### 计费与成本管控
| 模式 | 规则 | 优势 |
| --------- | --------------------------- | ---------- |
| 官方 | 服务费5% | 透明 |
| 中转 | 通道+服务（低于官方10%） | 易用 |

套餐：个人10元起、团队99元/月；管控：预警、分配、报表、优化推荐。
#### 用户与权限管理
手机号/微信登录；团队邀请/角色；企业分级（P1）。
#### 运营闭环系统
| 模块 | 功能 | 价值 |
| ---- | ------------------------- | ------------ |
| 会员 | 权益区分 | 付费需求 |
| 推广 | 邀请返利5% | 拉新 |
| 营销 | 签到/优惠 | 转化 |
| 统计 | 使用排行 | 决策 |

#### 数据安全与合规
加密传输/存储；脱敏；本地化；私有化Docker/宝塔（≤10min部署）。
#### 多厂商 API 对接配置中心
布局：顶部开关、左侧分类、中间标签、右侧帮助。
配置：差异化表单（海外/国内/多模态）；中转一键/自定义；校验/测试/日志。

### 非功能性需求
#### 性能
响应≤500ms；并发1000 QPS；兼容浏览器/H5；资源≤100M。
#### 稳定性
可用≥99%；备份每日；防限流切换。
#### 易用性
3步操作；简洁UI；FAQ/客服。
