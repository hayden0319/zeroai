## 1. 绘画设计功能

**模块优先级**：P0  
**对接模型**：Midjourney / DALL·E / Stable Diffusion（官方 API + 中转 API）

### 统一要求
- PC + 移动端 H5 全功能适配  
- 核心操作 ≤ 3 步  
- 生成内容云端永久保存 + 本地下载（无容量限制）  
- 支持模型一键切换对比

### 基础创作

| 功能点     | 详细操作 / 参数                                                                 | 性能指标               | 验收标准                              |
|------------|---------------------------------------------------------------------------------|-------------------------|---------------------------------------|
| 文生图     | 关键词（中/英）→ 风格（20+）→ 分辨率 512~4096 → 细节度 0-100 → 张数 1-10      | 1080P ≤ 30s<br>4K ≤ 60s | 无水印、无明显失真；风格符合 ≥ 95%   |
| 图生图     | 上传图 → 风格 → 相似度 0-100                                                    | 同上                    | 保留主要特征，风格迁移自然            |
| 扩图       | 上传图 → 方向（上下左右/全）→ 比例 1.2~2.0                                      | ≤ 40s                   | 衔接自然，无明显接缝                  |
| 局部重绘   | 上传图 → 涂抹区域 → 关键词                                                     | ≤ 20s                   | 重绘精准 ≥ 90%（未涂区域不变）        |

### 高级编辑

| 功能点     | 详细功能                                           | 性能指标         | 验收标准                          |
|------------|----------------------------------------------------|------------------|-----------------------------------|
| 图层操作   | 新建/删除/隐藏/顺序/透明度                         | ≤ 50ms           | 无卡顿、不丢失图层                |
| 特效添加   | 阴影/发光/模糊/渐变（可调参数）                    | ≤ 100ms          | 预览与实际一致                    |
| 字体编辑   | 500+ 字体 / 大小/颜色/加粗/斜体/下划线             | 加载 ≤ 300ms     | 中英文混排正常                    |
| 素材调用   | 1000+ 素材（图标/背景/装饰）→ 拖拽/缩放/旋转/搜索 | 加载 ≤ 300ms     | 按类型/风格搜索准确，无版权问题   |

### 输出格式

| 功能点     | 详细功能                                           | 性能指标           | 验收标准                              |
|------------|----------------------------------------------------|--------------------|---------------------------------------|
| 单张导出   | PNG（透明/普通）、JPG（80-100%）、SVG、PDF         | ≤ 1s               | 兼容 PS / AI / Figma                  |
| 批量导出   | 最多 50 张 / 次，打包 ZIP                          | 50 张 ≤ 10s        | 无损坏，ZIP 完整                      |
| 云端保存   | 按时间/风格/类型分类，支持关键词搜索               | 搜索 ≤ 300ms       | 无容量限制，搜索准确                  |

## 3. 计费管控功能

**模块优先级**：P0（商业核心）

### 定价体系
- **按量计费**：明码标价（示例：Midjourney 1080P ≈ 0.5 元/次）  
- **会员套餐**：
  - 免费版：每日 5 次基础调用  
  - 基础版：99 元/月，100 次全功能  
  - 专业版：299 元/月，500 次 + 8 折  
  - 企业版：999 元/月，2000 次 + 专属客服  
- **成本预估**：生成前显示费用 → 确认后扣费（误差 ≤ 1%）

### 充值支付
- 支持：微信 / 支付宝 / 银行卡 / 对公转账  
- 优惠：充 100 送 20、充 500 送 150、充 1000 送 400  
- 发票：电子普票/专票，24 小时内发邮箱  
- 记录：永久保存，可筛选

### 限额预警与控制

| 功能点     | 详细功能                                   | 验收标准             |
|------------|--------------------------------------------|----------------------|
| 额度预警   | 达 80% → 弹窗 / 短信 / 邮件                | ≤ 3 秒               |
| 超支控制   | 用尽后停止调用（可设自动充值）             | 准确率 100%          |
| 消费统计   | 按功能/模型/时间 → 饼图/柱状图             | —                    |
| 退款政策   | 未使用额度 7 天内无理由退（扣 5% 手续费）  | ≤ 3 个工作日到账     |

## 4. 私有化部署功能

**模块优先级**：P0（企业核心）

### 部署方式
- 宝塔面板一键部署  
- Docker 容器部署  
- 企业内网物理机部署

### 部署流程
- 提供详细文档 + 视频教程  
- 企业版提供远程技术协助  
- 目标时间：宝塔 ≤ 5 分钟，Docker ≤ 10 分钟

### 离线使用
- 断网后仍可调用已配置模型  
- 数据完全不出企业  
- 支持本地模型或缓存结果

### 版本更新
- 在线更新（需联网）  
- 离线更新包（U 盘/内网）  
- 更新保留用户数据、配置、历史

### 验收标准
- 宝塔部署 ≤ 5 分钟  
- Docker 部署 ≤ 10 分钟  
- 离线模式核心功能完整可用  
- 更新前后数据完整性 100%
